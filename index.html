Перевірка та вивчення кожного елемента форми

Form - Головний (батьківський) блок форми, записується так:

<form class="form" enctype="multipart/form-data" action="#"><form>

де Атрибут enctype - це кодування даних під час відправки на сервер. Для передачі файлів використовуємо - multipart/form-data

Атрибут action - "адреса" сервера. Не може бути пустим, варто ставити заглушку (#).

Атрибут method - метод передачі даних GET (за замовченням) - безпосередньо в адресному рядку; POST - приховано

input - Елемент вводу даних

<input class="form-test-form-page__input" id="name-01" autocomplete="off" placeholder= "Введіть ваше ім'я" name= "name" type="text">

де Атрибут type - тип елементу вводу даних

Атрибут name - ім'я поля (даних)

Атрибут value - значення поля (дані)

Атрибут placeholder - підпис поля

Атрибут autocomplete - дозволяє (за замовченням) / забороняє (значення off) автозаповнення поля

Атрибут required - обо'язково для заповнення

Атрибут tabindex - керує навігацією за допомогою TAB

Атрибут checked (для деяких типів) - вибраний за замовченням

label - Окремий підпис поля вводу

<label class="form-test-form-page__label" for="">

рядковий тег

де Атрибут for - закріплення тексту до input
